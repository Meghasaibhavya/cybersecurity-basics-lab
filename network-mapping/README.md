# Network Mapping

## What is network mapping?
Network mapping is the process of exploring, identifying, cataloging, and visually representing all devices (both real and virtual, local and remote) connected to your network.

## Tool Used
Kali Linux

## What I did
- Ran a basic Nmap scan on the target machine
- Identified open ports and running services
- Explored network hosts and their IP addresses

## What I learned
- Learned how to identify open ports and running services
