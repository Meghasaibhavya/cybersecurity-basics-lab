# Cybersecurity Basics Lab

A hands‑on cybersecurity lab repository documenting basic scans and network analysis. This repository contains beginner-level practice on basic networking concepts using tools like Nmap and Wireshark. It also includes introductory phishing analysis.

I documented my learning and hands-on practice while studying cybersecurity and preparing for internships. Kali Linux was used for Nmap and Wireshark.

## Topics Covered
- Network scanning and mapping using Nmap
- Network traffic analysis using Wireshark
- Phishing analysis and common red flags

## Directory Overview

- `network-mapping/` – Contains Nmap scans and documentation for network reconnaissance.
- `phishing-analysis/` – Notes and examples for identifying phishing techniques.
- `scanning-with-wireshark/` – Wireshark captures and analysis.
